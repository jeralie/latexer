For a random variable $X$ with expectation $\E(X) = \mu$, the variance 
of $X$ is:
\[\var(X) = \E((X - \mu)^2)\]
The square root of $\var(X)$ is called the standard deviation of $X$
\vspace{2 mm}
\\
\hline
\end{tabu}
}

\textbf{Theorem}: For a random variable $X$ with expectation $\E(X) = \mu$ and a 
constant $c$,
\[\var(X) = \E(X^2) - \mu^2\]
\[\var(cX) = c^2 * \var(X)\]
\vspace{2 mm}
